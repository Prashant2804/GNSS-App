# Demo Script

## 5-minute operator demo flow
1. `make run-pi-local` to start the Pi gateway in mock mode.
2. Launch Android app and open Connect tab.
3. Tap Connect (mock mode) and verify telemetry updates.
4. Create a project and open Collect tool.
5. Store a point with code "BM".
6. Open Stakeout and select the point; verify deltas.
7. Export project to CSV and verify file content.
